# taskmanager
A RESTful API for task management built with Go, Gin, and PostgreSQL

1.Database Setup
2.Task Model
3.Auto Migration
4.Full CRUD endpoints
5.Fixed update(PUT) behavior 

Clone the repo:
git clone https://github.com/Pong-Kantapon/taskmanager.git
cd taskmanager

Install dependencies:
go mod tidy

Run the API:
go run main.go
